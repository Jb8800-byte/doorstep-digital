# Apex Fitness - Portfolio Mock Website

A modern, bold fitness center website showcasing classes, training, and membership options.

## 🎯 Purpose

This is a **portfolio demonstration** site created to showcase web development capabilities for potential clients in the Miami fitness industry.

## ✨ Features

### Design
- **Dark, energetic aesthetic** - Bold colors with high contrast
- **Fully responsive** - Mobile, tablet, desktop optimized
- **Modern typography** - Inter font family
- **Dynamic animations** - Smooth scroll effects and transitions
- **Hero-driven layout** - Strong visual impact

### Technical
- **Single-file HTML** - Embedded CSS for simplicity
- **Font Awesome icons** - Professional iconography
- **Google Fonts** - Inter typeface
- **Responsive design** - Mobile-first approach
- **Smooth scrolling** - Enhanced navigation experience

## 📦 Structure

- `index.html` - Complete website (42KB, 1498 lines)
- All styles embedded in `<style>` tag
- External dependencies: Font Awesome, Google Fonts (Inter)

## 🚀 Usage

### View Locally
1. Open `index.html` in any modern browser
2. No build process or server required

### Deploy
Works with any static hosting:
- Netlify / Vercel (drag & drop)
- GitHub Pages
- Traditional web hosting

## 🎨 Customization for Real Clients

Easy to adapt for actual gyms:

1. **Branding** - Update colors in CSS variables
2. **Content** - Replace class schedules, trainer bios, pricing
3. **Images** - Add real gym photos and trainer headshots
4. **Forms** - Connect to email/CRM service
5. **Social** - Link real social media accounts

## 💡 Selling Points for Clients

- "Bold, energetic design that motivates action"
- "Mobile-optimized for on-the-go sign-ups"
- "Class schedules and trainer profiles built-in"
- "Clear membership CTAs throughout"
- "Professional design that builds credibility"
- "Fast-loading and search engine friendly"
