# La Costa Restaurant - Portfolio Mock Website

A fully responsive, modern restaurant website showcasing coastal cuisine in Miami.

## 🎯 Purpose

This is a **portfolio demonstration** site created to showcase web development capabilities for potential clients in the Miami restaurant industry.

## ✨ Features

### Design
- **Modern, clean aesthetic** with coastal color palette
- **Fully responsive** - works on desktop, tablet, and mobile
- **Smooth animations** and scroll effects
- **Professional typography** (Playfair Display + Poppins)
- **Intuitive navigation** with sticky header

### Sections
1. **Hero** - Eye-catching landing with CTAs
2. **About** - Brand story and key features
3. **Menu** - Signature dishes with pricing
4. **Gallery** - Visual showcase (placeholders ready for images)
5. **Hours & Location** - Business info
6. **Reservations** - Interactive contact form
7. **Footer** - Quick links and social media

### Technical
- **Vanilla JavaScript** - No framework dependencies
- **CSS Grid & Flexbox** - Modern layout techniques
- **Custom animations** - Smooth, professional feel
- **Mobile menu** - Touch-friendly navigation
- **Form validation** - User-friendly reservation system
- **Performance optimized** - Fast load times

## 🚀 Usage

### View Locally
1. Open `index.html` in any modern browser
2. No build process or server required

### Deploy
Works with any static hosting:
- Netlify / Vercel (drag & drop)
- GitHub Pages
- Traditional web hosting

## 🎨 Customization for Real Clients

Easy to adapt for actual restaurants:

1. **Branding** - Replace colors in CSS `:root` variables
2. **Content** - Update text, menu items, hours
3. **Images** - Replace emoji placeholders with real photos
4. **Contact** - Connect form to email/CRM service
5. **Social** - Add real social media links

## 📱 Responsive Breakpoints

- **Desktop**: 1200px+
- **Tablet**: 768px - 1199px
- **Mobile**: < 768px
